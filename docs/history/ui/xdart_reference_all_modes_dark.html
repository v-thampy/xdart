The four center-panel modes

The shell (left Data Browser, right wrangler cards, run dock) is identical across modes — only the top toolbar, the plot area, and which right-panel controls are greyed change. Each frame below shows one mode's center panel; the note says what the right panel does in that mode. Full window + both palettes: see the two _int2d reference files.
